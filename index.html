Projetos em Ciência de Dados

Construção de um programa para previsão de vendas futuras.

Eu usei Python, Estatística e técnicas de Machine Learning supervisionadas, como regressão, para gerar insights de vendas com base em suas características de performance de compra. Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.   • Python, Pandas
